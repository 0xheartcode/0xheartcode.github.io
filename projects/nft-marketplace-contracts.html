PROJECT OVERVIEW

A robust and secure NFT marketplace implementation with multi-token support, designed to handle both ERC721 and ERC1155 tokens with configurable currency options.

The platform features a comprehensive bidding system with customizable durations, royalty support via ERC2981, and advanced administrative controls for marketplace governance.
